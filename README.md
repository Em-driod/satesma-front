# Satesma Fountain Ventures - Frontend

This contains the frontend application for Satesma Fountain Ventures farm management system.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

The application will be available at http://localhost:3000
